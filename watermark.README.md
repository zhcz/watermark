# watermark
- [在线纯前端图片加水印](http://watermark.dxcweb.com/)
- 最安全，最快速的纯前端图片加水印，拒绝上传保证个人信息安全。
- 纯前端canvas加水印，开源代码

## 主要用途
在各种证件上添加“仅用于办理XXXX，他用无效。”，防止证件被他人盗用！<br/>
[新闻：身份证复印件被盗用所造成的损失，你想象不到！](https://www.weixin765.com/doc/reeodiqf.html)
